# Day 64 — §14.8 Adaptive Learning-Rate Methods (적응 학습률 3형제: 손잡이를 좌표별로 자동 조정)

Day63(§14.7)에서 **SGD·미니배치·상수스텝 망각**을 편향-분산 관점으로 묶었다. 거기서 학습률은
여전히 **모든 좌표에 하나**였다. §14.8 은 그 하나의 손잡이를 **좌표별로 자동 조정**하는 세 방법 —
**AdaGrad → RMSProp → Adam** — 으로 확장한다.

$$\text{단일-}\eta\ \text{SGD}\ \xrightarrow{\text{좌표별 누적}}\ \text{AdaGrad}
\ \xrightarrow{\text{누적}\to\text{감쇠기억}}\ \text{RMSProp}
\ \xrightarrow{+\text{모멘텀}+\text{바이어스보정}}\ \text{Adam}.$$

> **공통 축**: §14.7 과 똑같이 뿌리는 **편향-분산 + 전처리(preconditioning)** 다.
> AdaGrad 는 스케일 불일치를 상쇄하지만 단조 누적으로 죽고, RMSProp 은 감쇠기억 $\rho$(편향-분산 손잡이)로
> 유효 스텝을 살리며, Adam 은 모멘텀과 바이어스 보정으로 드리프트 추적과 초기 안정성을 함께 얻는다.

## 학습 목표
- **AdaGrad** 의 좌표별 $1/\sqrt{G_{t,j}}$ 가 ill-conditioned 스트림에서 **자동 전처리**로 작동함을 보이고,
  $G$ 의 단조 누적이 유효 스텝을 $\mathcal O(t^{-1/2})$ 로 죽이는 **장기 약점**을 확인한다.
- 누적합을 EWMA $v_t=\rho v_{t-1}+(1-\rho)g_t^2$ 로 바꾼 **RMSProp** 이 비정상 스트림을 추적함을 보이고,
  $\rho$ 를 **기억길이 $N_{\text{eff}}=1/(1-\rho)$ = 편향-분산 손잡이**로 해석(느린↔빠른 드리프트)한다.
- **Adam** = 모멘텀(1차 EWMA) + RMSProp(2차 EWMA) + **바이어스 보정**을 구현하고, 드리프트 스트림에서
  SGD·RMSProp 과 best-vs-best 비교, 그리고 바이어스 보정이 **초기 과도구간**에 하는 일을 정량화한다.

## 한 줄 정리
> **적응 학습률 = "학습률 손잡이를 좌표별로 자동 조정"** — AdaGrad(전처리·단조 누적으로 죽음),
> RMSProp(감쇠기억 $\rho$ 로 부활·$\rho$ 는 편향-분산 손잡이), Adam(모멘텀+바이어스 보정으로 추적·안정)은
> 모두 **하나의 편향-분산·전처리 이야기**이며, ill-conditioned/비정상 스트림에서 Adam 이 가장 견고하다.
