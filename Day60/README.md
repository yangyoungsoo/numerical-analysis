# Day 60 — §14.4 Transfer Case Study (전이 사례연구: 완성된 파이프라인을 새 도메인으로)

Day57–59(§14.1–14.3)에서 세운 파이프라인 —
**모델 선택 → 예측 + 불확실성 → 하이퍼파라미터 조정** — 을,
이번 §14.4에서는 **성질이 다른 새 도메인(target)** 으로 옮긴다.

$$\text{source에서 튜닝한 파이프라인}\;\xrightarrow{\;\text{전이}\;}\;\text{target에서 재조정}.$$

> **공통 축**: 전이는 "구조를 재사용하되 손잡이는 다시 맞추는 것" 이다.
> 원자료의 **참함수·잡음구조·표본밀도**가 바뀌면 세 결정(복잡도·불확실성·하이퍼파라미터)이 모두 재조정되어야 한다.
> 관통하는 뿌리는 여전히 **편향-분산 트레이드오프**와 **불확실성의 정량화**다.

## 학습 목표
- source에서 고른 **모델 복잡도**를 target에 그대로 이식(frozen)하면 편향-분산 균형이 깨짐을 보이고, CV/AIC/BIC로 **재선택**해 일반화오차를 회복한다.
- 균일분산 가정으로 만든 **예측구간(PI)** 을 이분산 target에 이식하면 피복률이 무너짐을 진단하고, 분산함수 $\sigma^2(x)$ 를 재추정(log-$\chi^2$ 편의 보정 포함)해 피복률을 회복한다.
- source의 튜닝 이력으로 **warm-start 베이지안 최적화**를 수행하고, 도메인 유사도에 따라 **양의 전이 / 음의 전이**가 갈림을 보인다.

## 문제 목록

| # | 노트북 | 주제 | 핵심 기법 | 핵심 결과 |
|---|--------|------|-----------|-----------|
| 1 | `CE_14_4_01.ipynb` | 모델 복잡도 전이 | 다항 LSQ, 5-겹 CV, AIC/BIC | source $d_S{=}5$, target $d_T{=}7$ 로 **선택 차수가 이동**; frozen($d_S$) 이식은 target 참함수 대비 RMSE 0.896, 재선택($d_T$)은 0.757 — **15.4% 개선** |
| 2 | `CE_14_4_02.ipynb` | 불확실성 전이 | 레버리지 PI, 분산함수 회귀 $\ln r^2\!\sim\!x$ + log-$\chi^2$ 편의 보정 | 균일분산 PI는 저잡음서 over-cover(0.985)·고잡음서 under-cover(0.805); 이분산 PI는 전 구간 **0.90 근처로 평탄**(좌 0.927·우 0.851, 평균 0.888) |
| 3 | `CE_14_4_03.ipynb` | 하이퍼파라미터 전이 | GP(RBF)+EI, grid/random/cold-BO/warm-BO | **near**: warm-start가 최소 regret(5.2e-4 < cold 7.7e-4), 초반부터 급강하 — 양의 전이. **far**: warm이 source 최적에 **정체**해 최종 7.8e-3 > cold 1.6e-3 — 음의 전이 |

## 한 줄 정리
> **전이 = "구조 재사용 + 손잡이 재보정"** — 복잡도는 target CV로 다시 고르고, PI는 target의 잡음구조로 다시 맞추며,
> 하이퍼파라미터는 warm-start로 아끼되 **음의 전이**를 경계해 target 데이터로 계속 갱신한다.
