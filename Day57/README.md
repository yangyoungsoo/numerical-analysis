# Day 57 — §14.1 Applied Case Studies (응용 사례연구: 실데이터 적합·모델선택·정규화)

챕터 13(최적화)까지 교재 로드맵의 **한 바퀴가 Day56에서 마무리**되었다. Day57부터는
Day56 README 예고대로 **응용 사례연구(Applied Case Studies)** 로 넘어간다. 이번 §14.1은
수치해법을 *실제 잡음 데이터*에 적용할 때 반드시 따라붙는 세 가지 통계적 판단 —
**불확실성 정량화 → 모델 선택 → 정규화**— 를 하나의 흐름으로 묶는다.

> **공통 아이디어**: 실데이터에는 잡음이 있다. 그래서 "가장 잘 맞추는 것"만으로는 부족하고,
> *얼마나 믿을 만한가*(불확실성), *얼마나 복잡해야 하는가*(모델 선택), *어떻게 안정화하는가*
> (정규화)를 함께 답해야 한다. 편향-분산 트레이드오프가 세 문제를 관통하는 축이다.

## 학습 목표
- **비선형 최소제곱**(LM)으로 감쇠 진동 모델을 적합하고, 공분산 $\hat\sigma^2(J^\top J)^{-1}$ 에서
  파라미터 **표준오차**까지 산출한다.
- 다항식 **차수 선택**을 학습/검증 분할과 **AIC·BIC**로 수행해 과소적합↔과대적합의 균형점을 찾는다.
- **릿지(티호노프) 정규화** 경로를 추적하고 **$k$-겹 교차검증**으로 $\lambda^\star$ 를 선택,
  악조건 설계행렬을 안정화한다.

## 문제 목록
| # | 노트북 | 주제 | 핵심 기법 | 핵심 결과 |
|---|--------|------|-----------|-----------|
| 1 | `CE_14_1_01.ipynb` | 비선형 최소제곱 + 불확실성 | LM(`curve_fit`), 공분산→표준오차 | 감쇠 진동 4파라미터를 소수 셋째 자리까지 회복, 잔차 무작위·$s^2\!\approx\!\sigma^2$; $\lambda,\omega$ 는 SE 작고 위상 $\phi$ 는 SE 큼 |
| 2 | `CE_14_1_02.ipynb` | 모델 선택(편향-분산) | 다항 차수, train/val, AIC/BIC | 훈련오차↓단조 vs 검증오차 U자형; AIC/BIC가 최적 차수 일치, **BIC가 더 단순한 모델 선호** |
| 3 | `CE_14_1_03.ipynb` | 릿지 정규화 + CV | $(V^\top V+\lambda I)^{-1}$, 정규화 경로, 5-겹 CV | $\lambda$↑로 계수 수축·조건수↓; CV가 $\lambda^\star$ 선택, **OLS(deg12) 요동 → 릿지는 매끄럽게 $f$ 추종** |

## 한 줄 정리
> **응용 사례연구 = "잡음 앞의 세 가지 질문에 답하기"** — 얼마나 믿을 만한가(표준오차),
> 얼마나 복잡해야 하는가(검증오차·AIC/BIC), 어떻게 안정화하는가(릿지+교차검증).
> 세 답을 관통하는 축은 언제나 **편향-분산 트레이드오프**다.
