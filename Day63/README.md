# Day 63 — §14.7 Incremental SGD / Mini-batch Online Learning (확률적 경사하강: 손잡이를 매 샘플·매 좌표로)

Day62(§14.6)에서 **재귀 최소제곱(RLS)+망각인자**로 배치해를 매 샘플 $\mathcal O(p^2)$ 에 재배열했다.
§14.7 은 같은 온라인 학습을 **더 싼 1차(gradient) 관점**으로 다시 본다 —
정규방정식 대신 **확률적 경사하강(SGD)** 로, 손잡이가 **학습률 $\eta$·스케줄 $\alpha$·배치크기 $B$** 로 흩어진다.

$$\text{망각-RLS}\;(\mathcal O(p^2))\;\xrightarrow{\;\text{1차 근사}\;}\;\text{SGD}\;(\mathcal O(p))
\;\xrightarrow{\;\text{배치·스케줄·상수스텝}\;}\;\text{편향-분산의 세 얼굴}.$$

> **공통 축**: §14.6 과 똑같이 뿌리는 **편향-분산 트레이드오프**다.
> 여기서는 그것이 세 손잡이로 나타난다 — 스케줄 지수 $\alpha$(수렴↔점근잡음),
> 배치크기 $B$(스텝정확도↔스텝개수), 상수스텝 $\eta$(추적속도↔잡음). 셋 다 같은 U자다.

## 학습 목표
- 1-샘플 **SGD** 를 구현하고, **학습률 스케줄** $\eta_t=\eta_0/t^\alpha$ 와 **Robbins-Monro 조건**
  $\sum\eta_t=\infty,\ \sum\eta_t^2<\infty$ 가 수렴을 어떻게 좌우하는지 로그-로그로 확인한다.
- **미니배치**의 배치크기 $B$ 가 만드는 *그래디언트 분산 $\propto1/B$* ↔ *갱신당 비용 $\propto B$* 상충을
  고정 데이터예산에서 정량화해 최적 $B^\star$ 를 찾는다.
- **상수 학습률 SGD = 망각 학습기** 임을 보이고($N_{\text{eff}}\approx1/\eta$), Day62 의 드리프트 스트림에서
  SGD 와 forgetting-RLS 를 **best-vs-best** 로 비교한다.

## 한 줄 정리
> **SGD 온라인 학습 = "손잡이를 매 샘플·매 좌표로 미세조정"** — 학습률 스케줄(수렴), 배치크기(잡음↔비용),
> 상수스텝(망각)의 세 손잡이는 모두 **하나의 편향-분산 U자**이며, 저여기 비정상 추적에서는
> $\mathcal O(p)$ SGD 가 $\mathcal O(p^2)$ forgetting-RLS 보다 싸고 안정적이며 정확할 수 있다.
