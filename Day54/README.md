# Day 54 — §13.9 Derivative-Free Optimization (도함수 없는 최적화: Nelder–Mead · 패턴 탐색 · 강건성)

챕터 13(함수의 최적화) **무도함수 절**. Day53(§13.8 비선형 최소제곱: Gauss–Newton·LM)까지는
야코비안/기울기를 활용했다. 이번 절은 **기울기를 전혀 쓰지 않고 함수값 비교만으로** 최소점을 찾는
직접탐색(direct-search)법을 정면으로 구현한다. 세 노트북이 (1) Nelder–Mead 단체법, (2) Hooke–Jeeves
패턴 탐색·좌표강하, (3) 잡음·불연속에서의 강건성 비교를 차례로 공략한다.

> **공통 아이디어**: 도함수 $\nabla f$ 가 없거나(블랙박스) 믿을 수 없을 때(잡음·불연속), **함수값의 대소
> 비교**만으로 하강 방향을 찾는다. 비교는 잡음의 부호에 둔감하고 꺾임에서도 잘 정의된다.

## 학습 목표
- **Nelder–Mead 단체법**을 반사·확장·수축·축소 연산으로 직접 구현하고, Rosenbrock에서 SciPy 표준 구현과 함수호출·정확도를 비교한다.
- **Hooke–Jeeves 패턴 탐색**과 **순환 좌표강하**를 구현해, 사선 골짜기에서 *패턴 이동*이 좌표강하의 지그재그를 어떻게 깨는지 확인한다.
- 잡음·꺾임으로 유한차분 기울기가 무너지는 영역에서 무도함수법과 **BFGS**의 **수렴 성공맵·성공률**을 정량 비교하고, 무도함수법이 유리한 조건을 정리한다.

## 한 줄 정리
> **도함수 없는 최적화 = "기울기 대신 함수값 비교"** — Nelder–Mead(단체)와 Hooke–Jeeves(패턴)는
> 기울기를 못 믿거나 구할 수 없을 때(잡음·불연속·블랙박스) 강건하게 작동하지만, 그 대가로 함수호출이
> 많다. 매끄럽고 깨끗한 문제라면 기울기법이, 그렇지 않다면 직접탐색이 표준 선택이다.
