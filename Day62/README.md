# Day 62 — §14.6 Online / Incremental Learning (온라인·증분 학습: 손잡이를 상시 미세조정)

Day61(§14.5)에서 드리프트를 **감지**하고 *언제* 재보정할지의 트리거를 세웠다.
§14.6은 관점을 한 걸음 넓힌다 — 매 샘플 **연속적으로** 계수를 갱신하는 온라인 학습으로,
"언제 손잡이를 돌릴지"에서 "손잡이를 **상시 미세조정**"으로 이동한다.

$$\text{배치 최소제곱}\;\xrightarrow{\;\text{Sherman-Morrison}\;}\;\text{재귀 최소제곱(RLS)}\;\xrightarrow{\;\text{망각인자}\;}\;\text{비정상 추적·연속 재보정}.$$

> **공통 축**: 세 문제를 관통하는 뿌리는 여전히 **편향-분산 트레이드오프**다.
> 여기서는 그것이 **망각인자 $\lambda$ = 유효기억길이 $N_{\text{eff}}=1/(1-\lambda)$** 라는 하나의 손잡이로 응축된다 —
> 기억을 길게(큰 $\lambda$) 잡으면 매끄럽지만 둔하고, 짧게(작은 $\lambda$) 잡으면 빠르지만 요란하다.

## 학습 목표
- 배치 정규방정식을 Sherman-Morrison 랭크-1 갱신으로 재배열한 **RLS**를 구현하고, **무릿지 초기화**로 배치 OLS 와 기계정밀도 동치임을 보인다.
- 지수 **망각인자 $\lambda$** 를 넣어 시간에 따라 변하는 계수를 추적하고, $N_{\text{eff}}=1/(1-\lambda)$ 에 담긴 추적속도↔잡음내성의 U자 상충을 정량화한다.
- 온라인 망각-RLS 를 Day61 의 트리거 기반 배치 재학습과 같은 드리프트에서 비교해, 정확도-비용 상충의 두 얼굴을 드러낸다.

## 한 줄 정리
> **온라인 학습 = "손잡이를 상시 미세조정"** — RLS 로 배치해를 매 샘플 $\mathcal O(p^2)$ 에 정확히 재배열하고,
> 망각인자 $\lambda$(=$N_{\text{eff}}$)로 비정상 드리프트를 연속 추적하면, 감지기·저장창 없이도 트리거 재학습보다 낮은 예측오차를 얻는다.
