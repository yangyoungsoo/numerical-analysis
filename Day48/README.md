# Day 48 — §13.3 Constrained Optimization (제약 최적화)

챕터 13(함수의 최적화)의 마지막 절. 제약 조건이 붙은 최적화 문제를 다루는 세 가지
대표 기법 — **벌점법, 내부점법, 단체법** — 을 직접 구현하고 비교한다.

## 학습 목표
- 등식 제약을 **이차 벌점법**으로 무제약화하고, 그 대가인 **조건수 폭발**과 **증강 라그랑지안**의 해법을 이해한다.
- 부등식 제약을 **로그 장벽 내부점법**으로 다루며 **중심 경로(central path)** 와 **쌍대 간격**을 추적한다.
- 선형계획(LP)을 **단체법(simplex)** 으로 꼭짓점을 따라 풀고 `scipy.linprog` 와 대조한다.

## 문제 목록
| # | 노트북 | 주제 | 테스트 문제 | 핵심 결과 |
|---|--------|------|-------------|-----------|
| 1 | `CE_13_3_01.ipynb` | 이차 벌점법 + 증강 라그랑지안 (등식 제약) | min $x_1+x_2$ s.t. $x_1^2+x_2^2=2$ | $\|c\|\sim1/\mu$, $\kappa(H)\sim\mu$; AL은 작은 $\mu$ 로 $\|c\|\to0$, $\lambda\to-0.5$ |
| 2 | `CE_13_3_02.ipynb` | 로그 장벽 내부점법 (부등식 제약) | min $(x-2)^2+(y-2)^2$ s.t. $x+y\le2,\,x,y\ge0$ | 중심 경로가 내부에서 $(1,1)$ 로 수렴, gap $=3\mu\to3\times10^{-6}$ |
| 3 | `CE_13_3_03.ipynb` | 단체법 + scipy.linprog | max $5x_1+4x_2$ s.t. $6x_1+4x_2\le24,\,x_1+2x_2\le6$ | $(0,0)\to(4,0)\to(3,1.5)$, $z^\star=21$, scipy와 $10^{-16}$ 일치 |

## 한 줄 정리
> 제약 최적화는 "제약을 벌점으로 녹이거나(외부), 경계에 장벽을 세워 내부에 머물거나(내부점), 선형 구조면 꼭짓점만 훑는다(단체법)" — 문제 구조에 맞는 도구 선택이 핵심이다.
