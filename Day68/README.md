# Day 68 — §15.1 Multi-Armed Bandits (다중 무장 밴딧: 탐색·활용의 정량화)

Day57–67(§14.1–14.11)에서 **온라인 학습**의 손잡이 조율 파이프라인을 통합했다. §15.1 은 그 연장선의 **한 발짝 위** —
"각 스텝에서 어떤 행동을 시도할지"라는 **순차 의사결정**의 가장 순수한 형태, **정지 확률 밴딧(stochastic bandit)** 를 다룬다.
세 정책을 편향-분산·리그렛 축으로 정면 비교한다: **$\varepsilon$-greedy** (정적 탐색), **UCB1** (신뢰상한 기반 자동 감쇠 탐색),
**Thompson Sampling** (사후분포 표본 기반 확률적 탐색).

$$
\text{탐욕 (고착)}\ \xrightarrow{+\varepsilon\text{-랜덤}}\ \varepsilon\text{-greedy (선형 리그렛)}
\ \xrightarrow{\sqrt{2\ln t / N_k}}\ \text{UCB1 } (R_T = O(\log T))
\ \xrightarrow{\text{posterior 표본}}\ \text{Thompson Sampling}.
$$

> **공통 축**: **리그렛 = 탐색 비용 + 고착 비용** 의 총합. 좋은 정책은 시간이 흐를수록 탐색 강도를 **자동으로**
> 줄여 두 비용을 동시에 작게 유지한다. §14 의 편향-분산 이야기가 순차 의사결정 위에서 반복된다.

## 학습 목표
- **$\varepsilon$-greedy** 정책과 **순수 탐욕** 을 $K$-무장 가우시안 밴딧에서 Monte-Carlo 로 비교하고, $\varepsilon$ 을
  상수로 두면 반드시 **선형 리그렛**이 남는 반면, $\varepsilon = 0$ 은 잘못된 표본평균에 **고착**될 수 있음을
  누적 리그렛과 최적팔 선택비율로 정량화한다.
- **UCB1** 의 신뢰상한 $\hat\mu_k + \sqrt{2\ln t / N_k}$ 이 시간에 따라 탐색을 자동으로 감쇠시키는 원리를 이해하고,
  같은 인스턴스에서 UCB1 이 **대수 리그렛** $R_T = O(\log T)$ 을 실제로 달성함을 log 축 선형성으로 검증한다.
- **Thompson Sampling** 이 각 팔의 posterior 로부터 표본을 뽑아 자연스러운 확률적 탐색을 부여함을 구현하고,
  **잘 지정된 사전**은 UCB1 을 능가하지만 **잘못 지정된 사전**은 초기 리그렛 대가를 크게 치름을 관찰한다.

## 한 줄 정리
> **밴딧 = 순차 의사결정의 순수 원자.** 상수 탐색($\varepsilon$-greedy)은 반드시 선형 리그렛을,
> 시간 감쇠 탐색(UCB1·TS)은 대수적 리그렛을 준다. 좋은 사전은 UCB1 을 능가하지만, 잘못된 사전은
> 초기 리그렛 대가를 크게 치른다 — **탐색 강도의 자동 조율**이 알고리즘의 리그렛 계열을 결정한다.
