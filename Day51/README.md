# Day 51 — §13.6 Interior-Point Methods for Nonlinear Programming (내부점법)

챕터 13(함수의 최적화) **심화 절**. Day50(§13.5 SQP)에서 *등식 제약* NLP를 KKT 비선형계에 대한
뉴턴(=QP 부분문제)으로 푸는 법을 익혔다. 이제 **부등식 제약**을 정면으로 다루는 **내부점법**으로 넘어간다.
세 노트북이 같은 부등식 제약 NLP를 (1) 로그 장벽, (2) 원-쌍대 뉴턴, (3) 비교·중심경로의 세 관점으로 공략한다.

> **공통 테스트 문제**: `min f(x,y)=(x-2)²+(y-2)²  s.t.  c(x,y)=x²+y²-1 ≤ 0`
> 무제약 최소점 `(2,2)`는 단위원 밖 → 제약이 **활성(active)**.
> 해석해 `x*=(1/√2, 1/√2)≈(0.7071,0.7071), f*≈3.3431, λ*=2√2-1≈1.8284`.

## 학습 목표
- 부등식 제약을 **로그 장벽** $-\mu\ln(-c)$ 으로 무제약화하고, **중심경로** $x(\mu)$ 가 $\mu\to0$ 에서
  $x^\*$ 로, 자동 생성된 승수 $\lambda(\mu)=\mu/(-c)$ 가 $\lambda^\*$ 로 $\mathcal O(\mu)$ 수렴함을 확인한다.
- 여유변수 $s>0$ 와 완화 상보성 $\lambda s=\mu$ 를 도입한 **섭동 KKT 방정식**에 뉴턴을 직접 적용하는
  **원-쌍대 내부점법**으로, **쌍대간격** $\lambda s$ 를 최적성 척도 삼아 **국소 2차 수렴**을 얻는다.
- 장벽 감소율 $\sigma$ 의 trade-off, 장벽법 vs 원-쌍대법의 **총 뉴턴 스텝수** 비교, 그리고
  SQP(등식)와 내부점법(부등식)이 공유하는 **"KKT=뉴턴"** 통일 관점을 정리한다.

## 한 줄 정리
> **내부점법 = "부등식 제약을 장벽/상보성 완화로 매끄럽게 바꿔 KKT를 뉴턴으로 푸는 것"** —
> 중심경로를 따라 $x(\mu)\to x^\*$ 이며, 원-쌍대법은 쌍대간격을 척도로 소수 반복에 2차 수렴한다.
