# Day 49 — §13.4 Quadratic Programming & Duality (이차계획법과 쌍대성)

챕터 13(함수의 최적화) **심화 절**. Day48의 제약 최적화 3종(벌점·내부점·단체법)에 이어,
이차계획(QP)을 정확하게 푸는 두 정공법과, 그 승수의 의미인 **쌍대성·그림자가격**을 다룬다.

## 학습 목표
- 등식 제약 QP를 **KKT 선형계** 한 번으로 정확히 풀고, KKT 행렬이 왜 대칭 **부정부호(saddle)** 인지 이해한다.
- 부등식 제약 QP를 **활성집합법(active-set)** 으로 풀며 working set의 추가/제거와 **상보여유성**을 추적한다.
- LP의 **쌍대문제**를 세워 **강쌍대성**과 상보여유성을 검증하고, 쌍대변수를 **그림자가격**($\partial z^\star/\partial b$)으로 해석한다.

## 한 줄 정리
> 라그랑지 승수 λ(등식)·KKT 승수 μ(부등식)·LP 쌍대변수 y는 **모두 같은 그림자가격** — 제약을 완화할 때 목적값이 변하는 비율이며, QP는 KKT 선형계(등식)·활성집합 연쇄(부등식)로 정확히 풀린다.
